## Installation

```console
# clone the repo
$ git clone https://github.com/darwishzain/recluse-board.git

# change the working directory to sherlock
$ cd recluse-board

# install the requirements
$ python3 -m pip install -r requirements.txt
```

## Run Recluse Board
```
# Run Recluse Board from Command Line
$ cd src
$ python -u main.py
```
